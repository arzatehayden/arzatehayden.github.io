# Hayden Arzate's — Personal Website

Live site: https://arzatehayden.github.io

Personal portfolio and résumé site for Hayden Arzate, Computer Engineer.

---


## Built With

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (IntersectionObserver for scroll animations)
- No frameworks, no dependencies, no build step

---

## Sections

- **Hero** — Name, tagline, and links to résumé and contact
- **About** — Bio, location, education, and availability
- **Skills** — Languages, hardware/embedded, and software tools
- **Projects** — Featured work with stack tags
- **Contact** — Email, LinkedIn
